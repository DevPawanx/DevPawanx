| Section | What it does |
|---|---|
| **Animated Header** | `readme-typing-svg` rotates 4 lines with typing effect |
| **About Me** | YAML-style bio block + professional summary |
| **Tech Stack** | `skillicons.dev` icon grids + `shields.io` badges grouped by category |
| **Advanced Tools** | 4-column table grid with all 14 tools as large badges |
| **Roadmap** | Phase badges (Completed/In-Progress/Planned) + sub-tool badges + progress indicators |
| **Projects** | 2×2 table with title, description, tech badges per project |
| **GitHub Stats** | Stats card + Top langs + Streak + Activity graph (all live APIs) |
| **Collabs** | Colored badge buttons for collaboration areas |
| **Contact** | Clickable icon badges for GitHub/Discord/Email |
| **Credits** | Full icon credits + Sakshu1347 support credit |
